  mrb_value mrb_range_new(mrb_state * mrb, mrb_value start, mrb_value end, mrb_bool exclusive)

  




  

    

Initializes a Range.



If the third parameter is FALSE then it includes the last value in the
range. If the third parameter is TRUE then it excludes the last value in
the range.
